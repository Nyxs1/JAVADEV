## ğŸš€ Quick Start

### Prerequisites

Pastikan kamu sudah menginstall:

-   PHP >= 8.2
-   Composer
-   Node.js & NPM
-   Git

### Installation

1. **Clone Repository**

    ```bash
    git clone https://github.com/yourusername/javadev.git
    cd javadev
    ```

2. **Install Dependencies**

    ```bash
    # Install PHP dependencies
    composer install

    # Install Node dependencies
    npm install
    ```

3. **Environment Setup**

    ```bash
    # Copy environment file
    cp .env.example .env

    # Generate application key
    php artisan key:generate
    ```

4. **Database Setup**

    ```bash
    # Run migrations
    php artisan migrate

    # Seed database (optional)
    php artisan db:seed
    ```

5. **Build Assets**

    ```bash
    # Development
    npm run dev

    # Production
    npm run build
    ```

6. **Start Development Server**

    ```bash
    php artisan serve
    ```

    Aplikasi akan berjalan di `http://localhost:8000`
